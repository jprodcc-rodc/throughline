## Posting notes for Rodc

- Best window: Tuesday-Thursday, 9-11am Eastern. Avoid weekend posting (overhang).
- Post tweet 1 first; reply each subsequent tweet to the previous one to build the thread. Don't pre-batch — Twitter's algorithm dings batch-posted threads.
- Pause ~60 seconds between tweet posts. Lets engagement begin on each tweet individually.
- Pin tweet 1 to profile for 7-14 days after posting.
- If a tweet is over 280 chars due to autolinking, kill the parenthetical first, kill the ellipsis second. Do NOT kill the dates, the brand-name, or the "ChatGPT's fine" line.
- Quote-tweet replies to thoughtful responses; do NOT subtweet hostile responses. Block-and-move-on > engage-and-explain on Twitter.
- If the thread takes off (>500 likes on tweet 1 within 24h), DO NOT add a 9th tweet. The thread is the thread. Adding tweets after-the-fact reads as desperation. Save additional content for a follow-up thread next week.
