## Brand-coherence check (cross-spec)

All 5 specs honor:

- **§7 Decision 1 (white-box thinking cards)** — `card-real` renders actual extracted user-voiced fields (no paraphrase); `vault-recall-history` shows actual counts (transparency over politeness rule for `recall_count >= 50`); `card-dedup` preserves 4 fields in DB even when render simplifies (data not lost, only collapsed); `first-insight` quotes topic verbatim from card.topic field (no paraphrase); `active-recall-base` uses null-by-default discipline applied to recall (return 0 cards rather than wrong cards).

- **§7 Decision 3 (verb "bring back" not "surface")** — `active-recall-base` callout uses `⚡ 我把这个带回来了` / `⚡ I brought this back`; `rodix_system.md` Past thinking integration section explicitly bans meta-commentary phrases ("I see in your vault…", "looking at your records…", "based on what I remember…", "my memory shows…"); `vault-recall-history` uses passive verb "Recalled" (anti-spin) not "brought back" (avoids cross-surface copy collision with the active recall callout).
