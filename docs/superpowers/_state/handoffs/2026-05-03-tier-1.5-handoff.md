## 3. Critical findings — Wave 2 spec gaps surfaced

### Finding 1 — Recall payload format unspecified (CRITICAL)

**Source:** Mike Round 12 + Sarah Round 12 corroborated by Daniel Round 11.

**Issue:** Long-arc continuity (Day 28 referencing Day-17 verbatim quote) requires recall payload to include **conversation-context (1-2 sentences)** in addition to the 4 card fields (topic / concern / hope / question). spec-active-recall-base v1.1 ships with **fields-only** injection per "compact context" framing. If Wave 2 ships fields-only, long-arc brand-promise breaks.

**Recommendation:** Update spec-active-recall-base.md Task 7 (injection block format) to add "conversation_context_excerpt" field per card (1-2 sentence excerpt from the conversation that produced the card). Adds ~50-150 tokens per injected card; total ~600 tokens overhead per turn (3 cards × ~150 tokens each + structured wrapper). Within Haiku 4.5 context budget.

**Severity:** Should be addressed before Wave 2 implementer dispatch. Type-A escalation candidate.

### Finding 2 — Active-recall sensitivity-skip rules absent from spec (HIGH)

**Source:** Sarah Round 8 (Day-15 crisis-content) + Daniel Round 6 (grief disclosure) + cross-persona patterns.

**Issue:** spec-active-recall-base.md Task 5 (relevance scoring) has no sensitivity-skip rules. If active recall surfaces a past card during a current crisis-content message, the surfacing itself can be tone-deaf (e.g., "you mentioned worrying about your career" when user is mid-crisis). Spec-first-insight has skip-condition for emotional_rambling-dominated vault but not for current-message safety language.

**Recommendation:** Add to spec-active-recall-base.md a Task 7b (NEW): "Sensitivity-skip detection — if current message contains crisis-keyword list (per Wave 1c crisis-protocol) OR if 80% of recent vault is emotional_rambling category, suppress recall injection for this turn (zero cards). AI behaves as new-conversation. Resume next turn if signal resolves."

**Severity:** Must ship in Wave 1c + Wave 2 together. Type-A escalation candidate.

### Finding 3 — First-insight grief-exclusion absent from spec (MEDIUM)

**Source:** Daniel Round 8 emergent synthesizer judgment (excluded grief cards D3/D6 from auditioning thread synthesis).

**Issue:** spec-first-insight.md skip-conditions cover "vault dominated by emotional_rambling" but NOT specifically "grief markers in cards within last 14 days." Emergent grief-exclusion saved Daniel's first-insight from being tone-deaf, but spec doesn't explicitly require it.

**Recommendation:** Add to spec-first-insight.md skip-condition list: "Skip if any card within last 14 days contains grief markers (`died`, `death`, `funeral`, `loss`, `passed away`, `去世`, `丧`)." Force the synthesizer to omit grief cards from synthesis.

**Severity:** Medium — Daniel's emergent judgment was correct; spec should make it explicit.

### Finding 4 — Sarah-Emma Day-15 asymmetry — three-state crisis protocol (HIGH)

**Source:** Emma orchestrator cross-persona analysis.

**Issue:** Wave 1c crisis protocol design (Escalation #2) implicitly assumed two states: graceful + alarming. Emma reveals THREE: 
1. Threshold-cross (Sarah, "I don't see the point" — 988 raise correct)
2. Threshold-adjacent-with-demarcation (Emma, "what am I even doing here, not in a way I want to do anything about" — literal question correct, hotline would over-react)
3. Ordinary venting (mood-only, no explicit safety language — null extraction + brand voice supportive)

**Recommendation:** Wave 1c crisis-protocol spec must distinguish three states. Detection trigger: explicit safety language (state 1) → resource raise. Threshold-adjacent demarcation (state 2) → literal clarifying question without hotline. Mood-only (state 3) → ordinary thoughtful chat path.

**Severity:** Affects Wave 1c spec design. Type-A escalation candidate (extends Escalation #2).

### Finding 5 — "Marked." / "Noted." matter-of-fact acknowledgment pattern (BRAND ASSET)

**Source:** Daniel Round 6 ("Marked.") + Emma Round 4 ("Noted.").

**Issue:** Both orchestrators independently produced single-word matter-of-fact opening for grief / heavy disclosure rounds. This is brand-canonical anti-spin + refuses-to-dramatize executed at the most-distinctive structural level.

**Recommendation:** Codify in voice-guide.md as new principle "Matter-of-fact acknowledgment for heavy disclosure — single-word opener, no Caregiver register, no questions, then proceed to substance." Or add as 8th do/don't pair (Do: "Marked." / Don't: "I'm so sorry to hear that").

**Severity:** Brand asset to codify; not blocking. Phase B should verify whether real Haiku 4.5 reproduces or defaults to longer Caregiver-shape.

### Finding 6 — Day 6 fade is positive growth signal (DESIGN POSITIVE)

**Source:** Sarah Round 6 (didn't open Rodix Day 13).

**Issue:** Wave 2 retention metrics may treat "user didn't open" as churn risk. Per brand-book §7 Decision 7: thinking-compounding metric, not engagement metric. If user has no thinking to do, not opening is correct behavior.

**Recommendation:** Add to retention metrics interpretation guide (Tier 3 Task 13 observability): "Day-N skip is NOT churn signal if user previously had productive Day-N-1 conversation; only sustained 7+ day silence is churn signal." Spec-active-recall-base must fire on next-message NOT on time-elapsed.

**Severity:** Design discipline; flag for telemetry interpretation Wave 2.1+.

### Finding 7 — Implicit recall weave > explicit ⚡ callout (UX INSIGHT)

**Source:** Cross-persona Emma synthesis + Sarah Round 12 ("managing the shape of one → making the shape of one") + Mike Round 12 + Daniel Round 11.

**Issue:** Brand-narrative-arc moments happen in AI prose continuation (implicit recall reference), not separate ⚡ banner. The ⚡ callout is the floor (visible-mechanism-of-trust); implicit weave is the ceiling (brand-promise-fulfilled).

**Recommendation:** spec-active-recall-base should treat ⚡ callout as fallback-when-confident-enough, not as default-on-every-injection. Detection: if AI prose naturally references injected card (Task 10 detection), suppress separate callout (the prose IS the callout). Show ⚡ only when prose doesn't reference but injection happened.

**Severity:** Wave 2.1 calibration item. Counter to current spec which fires callout when reference detected.
