# Code of conduct

We follow the short-form
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In plain language

- Be respectful. Assume good faith. Critique code, not people.
- No harassment, personal attacks, or discriminatory remarks.
- Keep discussions focused on the project. Off-topic soapboxing,
  including in PR comments, will be asked to move elsewhere.
- If someone tells you to stop, stop. If a thread feels heated,
  take a breath before replying.

## Scope

This applies to every project space — GitHub Issues, Pull Requests,
Discussions, and any other channel used on behalf of throughline.

## Enforcement

Report incidents privately to the maintainer via a GitHub private
security advisory (see [`SECURITY.md`](SECURITY.md)) — it's the most
reliable confidential channel we have. The maintainer will
investigate, respond, and if warranted, take one of: warning,
temporary ban from project spaces, or permanent ban. Decisions are
documented in a private log and the reporter is kept informed of
the outcome.
