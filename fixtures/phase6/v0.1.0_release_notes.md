First public release of **throughline** — the LLM-conversation → self-growing personal knowledge flywheel.

## What's included

- **OpenWebUI Filter** — per-turn 3-tier recall gate (cheap gate → Haiku RecallJudge → slash bypass) with dual-layer injection guard and concept-anchor fast-path.
- **Refine daemon** — slice → refine → route → dedup → dual-write (formal + buffer) pipeline with Pack system (PTE example shipped).
- **RAG server** — FastAPI wrapping BAAI/bge-m3 embedding + bge-reranker-v2-m3, with freshness-weighted rerank and personal-context boost.
- **Personal Context cards** — 4-layer stack (Filter valve → reranker boost → `<topic>__profile.md` auto-build → optional FastAPI agent) that injects your real profile into answers without contaminating the public RAG index.
- **Concept anchors** — self-growing whitelist of vault entities for short-query RAG gating.

## Accepted limitations (documented, not bugs)

1. **Brainstorm-mode drift** — English "should we X / what if Y / give me N ideas" classifies as `mode=native` instead of `mode=brainstorm`. Use `/recall` for explicit RAG on brainstormy inputs. See [`docs/CHINESE_STRIP_LOG.md`](./docs/CHINESE_STRIP_LOG.md) for background.
2. **First-turn bare-pronoun cheap-gate absent** — "it / that one" as first-turn inputs fall through to the Haiku judge (~$0.003/turn cost accepted). Context-backed pronouns (second-turn+) route correctly.
